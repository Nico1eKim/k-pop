# Opener | K-pop 행사를 한 눈에

![header_3d](https://github.com/P1Z7/frontend/assets/103186362/2bb8ad8f-9c79-430f-b4aa-73be1b2a5a04)

**Opener**는 K-pop 팬을 위한 오프라인 행사 정보를 한 곳에서 쉽게 확인할 수 있는 웹사이트입니다. 각종 카페 이벤트부터 팬광고, 전시회 등 다양한 이벤트 정보를 등록하고, 한눈에 찾아보세요!

🎉 배포 URL: https://www.myopener.kr/

⏰ 개발기간

- `2024. 01. 20` ~ `2024. 02. 22` (Ver 1.0)
- `2024. 02. 23` ~ ing (Ver 2.0)

<br>

## 프로젝트 소개

- Opener는 모두가 K-pop 아티스트의 오프라인 행사 정보를 등록하고, 수정할 수 있는 웹 플랫폼입니다.
- 다양하게 열리는 여러가지 비공식 행사 정보를 한 곳에서 쉽고, 편하게 확인할 수 있는 공간을 제공하기 위해 제작되었습니다.
- 유저는 좋아요 기능을 이용해, 지도와 캘린더 UI로 여러 행사 정보를 한 눈에 쉽게 파악할 수 있습니다.
- [Opener에 대해 자세히 알아보고 싶다면?](https://myopener.notion.site/Opener-eb86265c64e841e09ad8f5a56b0961da)
